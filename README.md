# -CL5D-Cirrhosis-Gene-Prioritization-Pipeline
A lightweight, Colab-friendly pipeline for **gene variant prioritization in cirrhosis**, integrating the **CL5D model framework** with curated knowledge from **ClinVar** and **GWAS Catalog**.   The system automatically parses VCF files (real or simulated fallback), performs robust annotation, and ranks variants based on biological relevance.
